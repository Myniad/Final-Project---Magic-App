Common Search filters

You can also search using multiple methods in one go.
For example, t:merfolk t:legend in one line would search for legendary merfolk type cards.

You also capable of adding a minus(-) sign in front a method to remove them from a search.
For example, t:goblin -t:creature in one line would look for goblin cards that arent creatures.

Methods for searching by card type:

t=[card type here]

t:[Card type here]

type:[Card type here]


Methods for searching by card color:

Note that colors can be abbreviated: W=White, U=Blue, B=Black, R=Red, G=Green

c:[color]

color:[color]

Example: color:[UW] this returns back cards that are both blue and white
